# Zero Waste, Thoughtfully

A responsive three-page Front-End Web Development project by Harinivevektha
Anandan, an Applied AI & Analytics student at Singapore Polytechnic.

The site presents practical zero-waste habits through an editorial visual
direction, personal reflection and Singapore-focused context.

## Pages

- `index.html` — project introduction and zero-waste overview
- `personalaction.html` — personal habits, actions and goals
- `about_me.html` — creator story, motivation and project context

## Technologies

- Semantic HTML5
- Responsive CSS3
- Vanilla JavaScript
- Accessible navigation and reduced-motion support
- Optimised WebP images
